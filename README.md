   # Drone-to-Edge Computing Project 

Overview - This project demonstrates a drone-to-edge computing system where image processing tasks are offloaded from a drone (simulated using a mobile device) to an edge server. The system captures images, sends them to a server, processes them using OpenCV, and returns results in real time.

  
#FEATURES:
-  Capture images using a mobile device (drone simulation)
- Send images to an edge server using Flask
- Process images using OpenCV (panorama stitching)
- Real-time response from server
- Reduces computation load on the drone

#TECH STACK:
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Image Processing: OpenCV
- Communication: HTTP (Flask API)
